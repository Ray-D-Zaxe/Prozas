This repo consists of cli based applications for Windows made in C.

To run them directly from the cmd or terminal regardless of the current directory,
>> Search for "edit the system environmental variables" on windows taskbar search
>> Click on "Environment variables"
>> Under system variables, double click on path
>> add the path of this folder as new
>> click all the ok
