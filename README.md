<h2>This repo consists of cli based applications for Windows made in C.</h2>
<br>
<hr>
To run them directly from the cmd or terminal regardless of the current directory,
<ol>
    <li>Search for "edit the system environmental variables" on windows taskbar search.</li>
    <li>Click on "Environment variables".</li>
    <li>Under system variables, double click on path.</li>
    <li>Add the path of this folder as new.</li>
    <li>Click ok to all.</li>
</ol>
Now you just,
<ol>
    <li>Open Terminal or CMD.</li>
    <li>Type in the name of the executable (eg, <i>Hello</i> or <i>Hello_1 -v -t yoo</i>) and press enter</li>
</ol>
<br>
If you don't wanna edit the Environment variables, you can still run them by
<ol>
    <li>Open the terminal</li>
    <li>Change directories using 'cd' command to the Code folder (or Bin folder if it exists) of this repo</li>
    <li>type in the name of the executable after './' (eg, <i>./Hello</i> or <i>./Hello_1 -v -t yoo</i>)</li>
</ol>
